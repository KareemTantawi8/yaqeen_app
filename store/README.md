# Store Listing Assets — يقين (Yaqeen)

This folder contains the artwork required to publish the Yaqeen app on the
**Apple App Store** and **Google Play Store**. All assets are derived from the
brand logo at `assets/images/Yaqeen.png` and are regenerated by running:

```bash
python3 tool/build_app_icon.py
```

## Files

| File | Size | Used for |
| --- | --- | --- |
| `app_store_icon_1024.png` | 1024×1024 | App Store Connect *App Icon* (and Marketing Icon) |
| `play_store_icon_512.png` | 512×512 | Google Play Console *App icon* (high-res) |
| `play_feature_graphic_1024x500.png` | 1024×500 | Google Play Console *Feature graphic* |

> All store assets are flat PNGs **without an alpha channel** (App Store and
> Play Store reject transparent / rounded-corner uploads — both stores apply
> the rounded mask themselves).

## On-device launcher icons

Launcher icons for both platforms are generated by
[`flutter_launcher_icons`](https://pub.dev/packages/flutter_launcher_icons)
from `assets/icons/app_icon.png` (master) and the two adaptive variants in
`assets/icons/`. Regenerate them after any icon change with:

```bash
dart run flutter_launcher_icons
```

## Brand

- **Display name:** يقين (Yaqeen)
- **Primary color:** `#1F6B61` (deep teal)
- **Accent gradient:** `#104A43` → `#1F6B61` → `#348E82`
- **Logo:** white Arabic calligraphy "يقين" with open Quran + sparkles
